# Geofence Manager

This is the starting point for the Geofence Manager take home assignment. Feel free to customise this file to reflect the deliverable.

## Project setup

Install dependencies:
`npm install`  

Run dev server:
`npm run serve`

Run API mock server:
`npm run mock-api-server`

Run unit tests:
`npm run test:unit`

Run linters:
`npm run lint`
